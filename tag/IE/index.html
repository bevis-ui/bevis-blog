Стилизация ребенка ссылки при :hover

Темы: IEСSS
Thursday, June 18, 2009

Хочу описать решение с пылу с жару.

Есть структура:

<a class="b-link" href="" >
    Телефон в копилке
    <img class="b-img" src="image.png" alt=""/>
</a>

Есть стили к ней:

.b-img {
    display: none
}

.b-link:hover .b-img {
    display: inline
}

Известно, что второй селектор в ИЕ6 не отработает.
